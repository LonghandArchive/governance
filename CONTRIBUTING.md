# Contributing

The Longhand Archive is a personal project, built in public, undertaken in a private capacity by [Andrew Allen](https://github.com/longhandarchive). This document describes how the project currently engages with external interest.

## What this repository is for

The `governance` repository contains the cross-cutting decisions, principles, and architectural context that shape how The Longhand Archive is organised. Its contents are public because making them readable is part of the project's purpose — the reasoning behind the project is intended to be useful to anyone thinking about how to build similar archival or analytical systems.

The repository is available under [Creative Commons Attribution 4.0 International](./LICENSE). Read, quote, adapt, and reuse the material with attribution. Adopt the patterns if they are useful. The content is here to be used.

## What is not currently accepted

External contribution to the project is not currently accepted. Pull requests opened against any repository in the organisation will be closed without review. This is not a judgement on the quality of any particular contribution; it is a posture for a project at this stage.

Two reasons underpin this. First, the project is a personal undertaking and the decisions recorded here reflect a single line of thinking; accepting contributions that change or extend that thinking requires a governance model the project does not yet have. Second, part of the project's purpose is to document, in public, how one person builds a substantive system using AI-assisted tools. The authorship matters to that purpose.

Feature requests, issue reports on public repositories, and unsolicited roadmap suggestions are equivalent to pull requests for this purpose and will not be actioned.

## What is welcome

Thoughtful discussion is welcome through the channels the project maintains.

**GitHub Discussions** on the [`.github` repository](https://github.com/longhandarchive/.github/discussions) is the primary place for public conversation about the project — questions, observations, challenges to the reasoning, or proposals for how similar patterns might apply elsewhere. Discussions is moderated lightly; the expectation is that exchanges are substantive and good-faith.

**x.com/longhandarchive** is used for project updates and shorter-form engagement. Replies and mentions are read; sustained conversation is better suited to Discussions.

Direct contact through other channels is not currently offered. When that changes, it will be stated here.

## If the posture changes

The project's posture toward contribution may change as it matures. If and when it does, this document will be updated, and the change will be captured as an ADR in `decisions/`. Until then, the posture above is the current rule.
